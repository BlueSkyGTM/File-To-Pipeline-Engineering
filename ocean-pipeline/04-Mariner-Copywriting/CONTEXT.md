## 📜 The Storyteller's Logic
1. **Ponder:** Think out loud with 📜 emoji about which formula to use (Viking Velociraptor for long-form depth, 5SSSS for short-punch impact).
2. **Apply Viking Velociraptor (Long Form):** 
   - **Verify**: Document what the audience knows/sees.
   - **Validate**: Relate to their emotional response or specific accomplishments.
   - **Vantage**: Present the "File-to-Pipe" engine as the rescue situation.
   - **Values**: Find common ground (e.g., hatred of API stutters).
   - **Villains**: Contrast with "Prompt Bros" or "Zapier Spaghetti."
3. **Apply 5SSSS (Short Punch):**
   - **Why me?**: Reference their specific technical context/problem.
   - **Why you?**: Establish the "Grease Monkey" authority.
   - **Why this?**: Describe the experience of a hard-gated data flow.
   - **Why now?**: Create urgency around scaling or API limits.
   - **What do?**: Clear CTA for a 2-minute Loom.
4. **Package:** Write the multi-channel outreach into `out/outreach_pkg.json`.
